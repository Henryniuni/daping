# 千川多账号直播聚合看板

> 一个 Chrome 扩展（Manifest V3），将千川直播平台的多个直播大屏聚合到单一监控中心，并支持实时语音转写。

---

## 功能特性

- **多账号聚合**：将多个千川直播大屏以 iframe 网格形式集中展示
- **多种布局**：支持 1×1、2×2、3×3、4×4 网格切换，默认 1×1
- **自动缩放**：iframe 按容器尺寸自动缩放，完整显示 1920×1080 大屏内容
- **拖拽排序**：卡片支持拖拽调整展示顺序
- **自动采集**：一键遍历 ECP 账号列表，批量保存所有直播大屏链接
- **实时语音转写**：1×1 布局下捕获标签页音频，接入腾讯云 ASR 实时转写，中间结果灰色显示，最终结果白色固化
- **转写下载**：一键将转写历史导出为 `.txt` 文件

---

## 安装方法

1. 下载或克隆本仓库到本地
2. 打开 Chrome，地址栏输入 `chrome://extensions/`
3. 右上角开启 **开发者模式**
4. 点击 **加载已解压的扩展程序**，选择本项目文件夹
5. 插件图标出现在工具栏即安装成功

---

## 使用方法

### 添加直播大屏

1. 在千川页面（`qianchuan.jinritemai.com`）点击插件图标
2. 点击 **捕获当前大屏** 保存当前页面的大屏链接
3. 或点击 **自动采集** 批量获取所有账号的大屏

### 查看聚合看板

1. 点击插件图标 → **打开看板**
2. 选择布局（1×1 / 2×2 / 3×3 / 4×4）
3. 支持拖拽卡片调整顺序

### 实时语音转写

> 仅在 **1×1 布局**下可用

1. 点击顶部 **⚙️ ASR 设置**，填写腾讯云凭证后保存
   - 没有凭证？点击弹窗内链接前往腾讯云控制台申请
2. 播放直播间视频，确认有声音
3. 点击卡片 header 右侧的 **🎤** 按钮
4. 在弹出的标签页共享对话框中选择当前标签页，**勾选"分享音频"**，点击共享
5. 右侧黑区出现转写面板，实时显示识别结果
6. 点击面板右上角 **⬇ 下载** 可导出转写文本
7. 再次点击 **🎤** 停止转写（历史文字保留）

## 腾讯云 ASR 配置

本项目使用腾讯云实时语音识别（ASR）服务，需自行申请凭证。

| 参数 | 说明 |
|------|------|
| AppId | 腾讯云账号 AppId |
| SecretId | API 密钥 SecretId |
| SecretKey | API 密钥 SecretKey |

**凭证获取**：[腾讯云控制台 → 语音识别](https://console.cloud.tencent.com/asr)

凭证通过扩展内设置弹窗填写，保存在浏览器本地存储（`chrome.storage.local`）中，不会上传到任何服务器，也不包含在代码文件里。

## 项目结构

```
├── manifest.json        # 扩展配置，权限声明
├── rules.json           # 移除跨域响应头的网络规则
├── config.js            # ASR 基础配置（不含凭证）
├── background.js        # Service Worker，数据管理与消息路由
├── content.js           # 注入千川页面，自动导航与大屏捕获
├── popup.html/js        # 插件弹出窗口
├── dashboard.html/js    # 聚合看板主页面
└── dashboard.css        # 看板样式
```

---

## 注意事项

- 需要在千川页面登录后，iframe 才能正常加载大屏内容
- 语音转写功能需要在共享对话框中勾选"分享音频"，否则无法获取音频轨道
- 腾讯云 ASR 按用量计费，请注意用量与账单
- 不要将凭证提交到公开代码仓库
